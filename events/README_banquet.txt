# Как запускать цикл банкетов (банкет-бот)

1. Для чего нужен этот цикл?
   - Автоматизация создания банкета основой (usssia_main)
   - Автоматизация участия всех твинов в банкете по заданному ID
   - Можно повторять цикл несколько раз подряд (например, для нескольких банкетов за вечер)

2. Где находится функция?
   - Файл: events/events.py
   - Функция: run_banquet_cycle(repeats=1)

3. Как вызвать цикл банкетов?
   - В любом месте вашего проекта (например, в отдельном скрипте):

     from events.events import run_banquet_cycle
     run_banquet_cycle(repeats=1)  # 1 — сколько раз повторить цикл

4. Как работает цикл?
   - Основой создаётся банкет (banquet.py)
   - Все твинки по очереди заходят и выполняют банкет (banquet_event.py)
   - Всё повторяется столько раз, сколько указано в repeats

5. Что нужно для работы?
   - Все картинки-кнопки должны быть в папке screenshots/
   - ID банкета (BANQUET_ID) должен быть актуальным (можно изменить в events/events.py)
   - Все функции перехода между аккаунтами, входа/выхода из игры должны быть реализованы

6. Важно!
   - Этот цикл НЕ связан с ежедневными игровыми обязанностями, он только для банкетов
   - Можно запускать отдельно, когда идёт акция или просто нужно провести банкет

7. Пример отдельного скрипта для запуска:

     # файл: run_banquets.py
     from events.events import run_banquet_cycle
     run_banquet_cycle(repeats=2)  # Запустить 2 банкета подряд

---

Если что-то не работает — проверьте, что все функции и картинки на месте, и обратитесь к этому файлу для подсказки!
